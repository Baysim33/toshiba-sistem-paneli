
# 🖥️ Toshiba Canlı Sistem Paneli & Siber Kalkan

Emektar Toshiba dizüstü bilgisayarlar ve **CachyOS (Arch Linux)** tabanlı sistemler için özel olarak geliştirilmiş, hafif (lightweight) ve anlık güncellenen bir terminal takip paneli.

## 🛠️ Kurulum ve Çalıştırma Steps

Projeyi bilgisayarınıza kurup çalıştırmak için sırasıyla aşağıdaki adımları terminalinizde uygulayın:

1. Projeyi Bilgisayarınıza İndirin (Clone)

İlk olarak bu depoyu kendi bilgisayarınıza çekin:

```bash
git clone https://github.com/Baysim33/toshiba-sistem-paneli



Proje Klasörüne Girin
​İndirilen klasörün içine terminal üzerinden geçiş yapın:

```bash
cd toshiba-sistem-paneli


Gerekli Kütüphaneyi Kurun
​Sistem bileşenlerini okumak için gerekli olan Python kütüphanesini yükleyin:

```bash
pip install psutil


Paneli Ateşleyin!
​Her şey hazır, paneli canlı olarak başlatmak için şu komutu yürütün:

```bash
python kontrol.py


🚀 Özellikler
​Anlık İşlemci (CPU) Takibi: Görsel barlarla CPU yükünü canlı izleme.
​RAM Kullanımı: Bellek tüketimini anlık olarak grafiksel raporlama.
​Siber Kalkan: Kritik portları tarayarak sızma girişimlerine karşı açık port uyarısı.
​Güvenli IP Gösterimi: Sistem IP adresini anında ekrana basma.
